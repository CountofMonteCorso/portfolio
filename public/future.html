Future Projects

Green Glass Door isn't a game I invented, but I never saw it online so I figured I'd do it some justice. The name of the game is figuring out how to play the game; once you figure out the riddle, you can't really play the game again. This game was made with vanilla JavaScript and uses an algorithm to read and evaluate user inputs to see if the response meets certain criteria. The logic is complete, but I'm still building out features for the game and have plans to rebuild it in React with even more features. You can play the most basic version by clicking on the image to the left though :]
